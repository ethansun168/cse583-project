# Generate dataset file
```
$ python3 json_generator.py
```
Requires `database_template.json` and `database/`
# Set up virtual environment
```
$ python3 -m venv env
$ source env/bin/activate
$ pip install -r requirements.txt
```
# Train the model
```
$ python3 train.py
```
# Run Tests
```
$ python3 run.py
```
